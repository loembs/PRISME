# Prisme — Le média qui éclaire l'actualité

Bienvenue sur le projet **Prisme Media**, une plateforme d'actualités et de documentaires innovante focalisée sur l'Afrique et les enjeux internationaux.

## À Propos

Prisme est conçu pour offrir une perspective unique sur l'actualité à travers des reportages de fond, des analyses percutantes et des décryptages vidéo modernes. L'interface est inspirée par les codes visuels du média Brut, offrant une navigation fluide et immersive.

## Technologies Utilisées

Ce projet est construit avec une stack moderne et performante :

- **Vite** : Pour un environnement de développement ultra-rapide et un build optimisé.
- **TypeScript** : Pour une base de code typée, sûre et maintenable.
- **React** : Pour une interface utilisateur dynamique et réactive.
- **shadcn/ui** : Pour des composants d'interface élégants et accessibles.
- **Tailwind CSS** : Pour un design système flexible et moderne.
- **Lucide React** : Pour une iconographie claire et cohérente.

## Installation et Lancement

Pour lancer le projet localement :

### Prérequis
- Node.js (version LTS recommandée)
- npm ou yarn

### Étapes
1. Clonez le dépôt.
2. Installez les dépendances :
   ```sh
   npm install
   ```
3. Lancez le serveur de développement :
   ```sh
   npm run dev
   ```
4. Accédez à l'application sur `http://localhost:8080`.

## Architecture du Projet

- `src/components` : Composants réutilisables (UI, layout, vidéo).
- `src/pages` : Pages principales de l'application.
- `src/data` : Données simulées (mock data) pour le contenu.
- `src/lib` : Utilitaires et configurations externes (ex: utils.ts).
